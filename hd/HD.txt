Bước 2: Kết nối hosting
Cài đặt package cho sublime tại dây : https://toidicode.com/cai-dat-va-su-dung-sftp-package-cho-sublime-text-92.html

Sau khi cài xong, copy file sftp-config.json vào thư mục cần tương tác, nếu file này tồn tại thì ghi đè.

Tìm đoạn code sau:
"host": "xxx",
 "user": "xxx",
 "password": "xxxx",
 "port": "22",
 "remote_path": "/xxxxxxxxxxx/",

thay đôi thông tin từ thông tin account của bạn đã cung cấp.
